# typescript-practice

Improve TypeScript skills.

## Setup

```bash
npm install
```

## Usage

### Node.js

Build TypeScript to JavaScript, then run:

```bash
npm run build   # compiles src/ → dist/
npm start       # runs dist/index.js with node
```

### Bun

Run TypeScript directly (no build step required):

```bash
bun run src/index.ts
# or via the npm script:
npm run dev
```
